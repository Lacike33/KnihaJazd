# Frontend - Kniha jázd

Next.js frontend aplikácia pre Kniha jázd.

## Technológie

- Next.js
- React
- TypeScript
- Tailwind CSS (plánované)

## Vývoj

```bash
npm install    # Inštalácia závislostí
npm run dev   # Spustenie dev servera
```

## Štruktúra

Bude vytvorená po inicializácii Next.js aplikácie.
